Apply Meta-Learning to Text classification

Model-Agnostic Meta-learning is a learning scheme which aims for training the model with a small number of samples, resulting in good generalization and fast adaptation ability of the model.

Meta-Learning computation consists of the first order term(gradient term) and the second order term (Hessian or gradient of gradient) which is computationally expensive.

Reptile algorithm is the one of the algorithm which overcome the drawback of second order computation by approximating the original algorithm into the first order MAML with the newly defined gradient or reptile gradient.

It has been observed that the reptile algorithm has a good performance on image classification task. In this project, we mainly focus on apply Reptile on Text classification.

CSAW’20 HackML Challenge

This project is meant to restore the backdoored models.

"As ML and AI systems are widely used and adopted across various domains, t hese models are prone to various attacks and vulnerabilities much like software systems. Recent works have shown that neural networks can contain stealthy backdoors that allow attackers to trigger malicious behavior. CSAW 2020 HackML competition challenges teams across the world to design and develop novel defense strategies and detection methodologies to mitigate powerful backdoor attacks on neural networks."


We tried several different ways to mitigate the backdoor.

1. Fine-pruning, 2. Filter via Mutul Info, 3.STIP, the overall performance is 90% up mitigation.
